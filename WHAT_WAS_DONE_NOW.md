# Що було зроблено зараз

## ✅ Реалізовано повну підтримку Front/Back фото в адмінці

### 1. Вибір типу фото при завантаженні
**Де:** Форма редагування товару → Секція "Фото товару"

**Що додано:**
- 4 кнопки вибору типу фото:
  - **Спереду** - основне фото товару
  - **Ззаду** - фото задньої частини
  - **Збоку** - фото збоку
  - **Додаткове** - інші фото та деталі

**Як працює:**
1. Відкриваєш товар на редагування
2. Вибираєш тип фото (за замовчуванням "Спереду")
3. Завантажуєш фото
4. Фото зберігається з правильним типом в базі даних

### 2. Вибір типу для AI генерації
**Де:** Форма редагування товару → Секція "Nanobanana — генерація фото"

**Що додано:**
- 3 кнопки вибору типу генерації:
  - **Спереду** - генерація виду спереду
  - **Ззаду** - генерація виду ззаду (дизайн на спині)
  - **Збоку** - генерація виду збоку (профіль)

**Як працює:**
1. Вибираєш тип генерації (за замовчуванням "Спереду")
2. Завантажуєш опорне фото
3. Вибираєш колір та стать
4. Ставиш галочку "Генерувати після збереження"
5. Зберігаєш товар
6. AI генерує фото з правильним ракурсом

### 3. Оновлено бекенд
**Файл:** `fastapi_app/routers/ai_photos.py`

**Що змінено:**
- Додано параметр `imageType` до ендпоінту `generate-multiple`
- AI тепер генерує фото з правильними промптами для кожного типу:
  - **Front:** "Full-body front view, model facing camera..."
  - **Back:** "Full-body back view, model facing away, showing back design..."
  - **Side:** "Full-body side profile view..."
- Згенеровані фото зберігаються з правильним типом

   - Вибери колір (можна клікнути на фото для вибору кольору)
   - Вибери стать моделі
   - Постав галочку "Генерувати після збереження"
3. Збережи товар
4. AI згенерує фото ззаду з дизайном на спині

6. Тепер товар має фото спереду (завантажене) та ззаду (згенероване AI)

## 📊 Що зберігається в базі даних

### Новий формат зображень:
```json
{
  "images": [
    {
      "url": "https://res.cloudinary.com/...",
      "type": "front",
      "order": 0,
      "isGenerated": false,
      "cloudinaryPublicId": "vinesent/products/..."
    },
    {
      "url": "https://res.cloudinary.com/...",
      "type": "back",
      "order": 1,
      "isGenerated": true,
      "cloudinaryPublicId": "vinesent/products/..."
    }
  ]
}
```

### Типи зображень:
- `front` - фото спереду (основне)
- `back` - фото ззаду
- `side` - фото збоку
- `additional` - додаткові фото
