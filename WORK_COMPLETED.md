### 2. API Endpoints

#### Новые эндпоинты:
```
GET    /api/v1/products/{id}/images          - Получить все изображения
POST   /api/v1/products/{id}/images          - Загрузить с типом
GET    /api/v1/products/{id}/images/{type}   - Получить по типу
DELETE /api/v1/products/{id}/images          - Удалить изображение
```

#### Обновленные эндпоинты:
```
POST   /api/admin/ai-photos/generate         - Добавлен параметр imageType
```

---

### 3. Формат данных

#### Новая структура изображений:
```json
[
  {
    "url": "https://res.cloudinary.com/...",
    "type": "front",
    "order": 0,
    "isGenerated": false,
    "cloudinaryPublicId": "vinesent/products/..."
  },
  {
    "url": "https://res.cloudinary.com/...",
    "type": "back",
    "order": 1,
    "isGenerated": true,
    "cloudinaryPublicId": "vinesent-ai/..."
  }
]
```

#### Поддерживаемые типы:
- `front` - фото спереди (основное)
- `back` - фото сзади (для дизайна на спине)
- `side` - фото сбоку (для силуэта)
- `additional` - дополнительные фото

Preserve all back design elements: logos, text, graphics, embroidery.
Maintain fabric texture and realistic wrinkles.
Neutral white background, soft diffused studio lighting, photorealistic, crisp detail.

Neutral white background, soft diffused studio lighting, photorealistic, crisp detail.
